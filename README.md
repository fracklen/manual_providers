# Couch reports


## Install couchapp

https://github.com/couchapp/couchapp

```bash
sudo apt-get install python-dev libxml2-dev libxslt-dev
curl -O https://bootstrap.pypa.io/get-pip.py
sudo python get-pip.py
sudo pip install couchapp
```
